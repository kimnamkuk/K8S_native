# 1. Deployments 는 무엇인가?
  - Stateless 한 상태의 Pod들의 집합입니다.
  - 정해진 Replicas 만큼 Pod 개수를 유지합니다.
  - 만약 Pod가 응답이 없을 경우 해당 Pod를 새로운 Pod로 변경합니다.
